### **6. JSON Functions**  
- `JSON_OBJECT(key, value, ...)`: Creates a JSON object.  
- `JSON_EXTRACT(json, path)`: Extracts a value from a JSON document.  
- `JSON_ARRAY(val1, val2, ...)`: Creates a JSON array.
